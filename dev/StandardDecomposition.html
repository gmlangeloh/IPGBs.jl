IPGBs.StandardDecomposition.AdmissiblePair — Type

Represents an admissible pair, i.e., (x^m, σ) where x^m is a standard monomial and σ is a set of indices disjoint from the support of m.

Additionally, for this pair to be standard, it has to be minimal w.r.t. the following order: (x^m, σ) <= (x^m', σ') iff x^m divides x^m' and supp(x^m' / x^m) ∪ σ' ⊆ σ.

The minimal elements w.r.t. this order are the standard pairs and they cover other admissible pairs.
