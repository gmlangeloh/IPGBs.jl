IPGBs.IPInstances.IPInstance — Type

Represents an instance of a problem

min C * x

s.t. A * x = b

0 <= x <= u

x in ZZ^n

The instance is stored in normalized form, with permuted variables so that the variables appear in the following order: bounded, non-negative but unbounded, unrestricted.

IPGBs.IPInstances.compute_permutation — Method
compute_permutation(bounded :: Vector{Bool}, nonnegative :: Vector{Bool}) :: Tuple{Vector{Int}, Int, Int}

Return a permutation of variables that puts the variables in the order [ bounded ; unbounded and restricted ; unrestricted ], along with the indices of the last bounded variable and the last unbounded but restricted variable.

This operation should be stable with respect to the initial ordering of variables.

A permutation is represented by a vector perm such that perm[i] = j means that variable i is sent to j by the permutation.
