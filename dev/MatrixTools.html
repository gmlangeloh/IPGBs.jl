IPGBs.MatrixTools.hnf_lattice_basis — Method
hnf_lattice_basis(A :: Matrix{Int})

Return a row basis for the lattice ker(A) computed using the Upper
Hermite Normal Form. The entries of this basis tend to be smaller than
those computed directly from kernel(A).

IPGBs.MatrixTools.normalize_hnf! — Method
normalize_hnf!(H :: Generic.MatSpaceElem{T})

Change H to an upper row HNF matrix satisfying the following property: all entries above a pivot are non-positive and of smaller magnitude than the pivot

Assumes H is already in HNF as defined by AbstractAlgebra.jl, that is, it is in upper row HNF satisfying:

  • all entries above a pivot are non-negative and of smaller magnitude than the pivot
