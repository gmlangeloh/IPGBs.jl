IPGBs.GBElements — Module

This module defines all the binomial data structures used in my implementations of Buchberger's algorithm and Signature-based algorithms.

IPGBs.GBElements.GBElement — Type

Abstract type used for GB computations. It is meant to generalize both binomials and binomials with signature, in order to simplify the implementation of reduction algorithms.

IPGBs.GBElements.model_truncation — Method

Checks feasibility of:

Ax = b - Av 0 <= x <= u

with x either a vector of real variables or integer variables, whatever is defined in model. In order to do this efficiently, updates the RHS of the previously built model by modifying constraints.

Returns true iff v is feasible for the above model, which means it should NOT be truncated.

IPGBs.GBElements.reduces — Method
reduces(
g :: P,
filter :: Vector{Int},
reducer :: T,
gb :: S;
fullfilter :: Bool = true,
negative :: Bool = false,
is_singular :: Ref{Bool} = Ref(false)

) :: Bool where {P <: AbstractVector{Int}, T <: AbstractVector{Int}, S <: AbstractVector{T}}

Checks whether reducer divides g, using the filter of reducer for efficiency. When fullfilter = true, checks if g.head >= reducer.head and g.tail >= reducer.tail coordinate-wise, while also checking a degree criterion.
