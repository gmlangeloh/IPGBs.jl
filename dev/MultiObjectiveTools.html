Some useful functions to deal with multi-objective optimization problems.

Changes direction of the given Pareto set by changing signs (i.e., transform a solution from a minimization problem into maximization and vice-versa).

IPGBs.MultiObjectiveTools.is_nondominated — Method

Checks whether the solution at index i of pareto is nondominated with respect to the rest of pareto.

Returns nothing in case the solution is nondominated, or an index pointing to proof that the solution is inefficient, otherwise.
