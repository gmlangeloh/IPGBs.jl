IPGBs.GBAlgorithms.GBAlgorithm — Type

A generic Gröbner Basis algorithm. Currently, it can be either a BuchbergerAlgorithm or a SignatureAlgorithm.

prepare_gb_output(algorithm :: GBAlgorithm) :: Vector{Vector{Int}}

Return the user-friendly output Gröbner Basis of algorithm, reduced, sorted and in the original variable ordering.

IPGBs.GBAlgorithms.reintroduce_truncated! — Method
reintroduce_truncated!(algorithm :: GBAlgorithm)

Add truncated elements of the given Markov basis back into the GB.

This is used at the end of the computation to keep the GB as a basis of the input ideal, regardless of truncation.
